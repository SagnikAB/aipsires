# 🚀 AI-Powered Smart Interview & Resume Evaluation System

An AI-driven web application that evaluates resumes and analyzes candidate suitability using scoring algorithms and keyword intelligence.

---

## 🔥 Features

- 📄 Resume Upload (PDF / DOC / DOCX)
- 🤖 AI-Based Resume Scoring
- 🎯 Keyword & Skill Detection
- 📊 Analysis History Tracking (SQLite)
- 🌗 Dark / Light Mode
- 🖥 Modern SaaS UI
- ⚡ Fast & Lightweight Flask Backend

---

## 🛠 Tech Stack

- Python
- Flask
- SQLite
- HTML / CSS / JavaScript
- NLP-based scoring logic
